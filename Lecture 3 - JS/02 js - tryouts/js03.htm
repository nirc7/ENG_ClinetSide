3. Functions:

The special functions features are:
  • no MAIN - no MAIN function; the code is perforemed as it is seen
  • no declaration - the function is implemented without prior declaration
  • no typing - the arguments and return values types need not declared in function implementation
  • two parted code-
    • functions implementations - not performed until the function is called
    • other code - performed when met
