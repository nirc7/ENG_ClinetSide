13. Strings

The String object serves as built-in library of string functions.
No need to create an instance of it or to include or import it explicitly as it is invoked whenever a string is used.
It is possible however to create a new String explicitly. There is a slight difference in the behaviour of a string if it is created implicitly as a literal or an object, but that is very seldom.

Syntax to create a string:

s1 = "foo" // creates a string literal value
s2 = new String("foo") // creates a String object
