1. The minimal program:

After introduction we learn how to write and deploy a JS program.

JS program has to be embedded in HTML page using the SCRIPT tag. Code is everything that is between opening SCRIPT tag ( <script> ) and closing SCRIPT tag ( </script> ).

More than one SCRIPT element is allowed. It is recommended to put it in the HEAD section of the document, so that it will be available to all elements, as the HEAD is loaded before the BODY. (This will not work if the script refers to an element before it is loaded, and in that case it should be inserted after its declaration and loading).
