24. Window object


 

Window


The window is one of the most important objects, in a sense the root of the browser objects tree. It is the parent of document object. Some of its methods and properties are very usefull.
Usually we do not encounter it as the browser is waiving the need for the header "window.", e.g.
"window.location" is written by us usually as "location",
"window.alert" is written by us usually as "alert", etc.
