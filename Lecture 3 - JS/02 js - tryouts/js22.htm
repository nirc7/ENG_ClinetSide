22. Date predifined class


 

Some methods of Date

Methods Explanation
Date() Returns a new Date object
getDate() Returns the day of the month from a Date object. (1-31)
getDay() Returns the weekday. (0-6)
getMonth() Returns the month. (0-11)
getFullYear() Returns the year. (2000)
getHours() Returns the hour as a value between 0 and 23
getMinutes() Returns the minute. (0-59)
getSeconds() Returns the second. (0-59)
