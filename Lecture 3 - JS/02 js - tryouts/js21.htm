21. Class variables and prototype


 

Prototype

Prototype creates a "template" or prototype of an object. This does not create an actual instance of the object, but defines the structure of the object. the usage is
ClassName.prototype.propertyName
or
ClassName.prototype.methodName


 

Class Variable

Class variable also called static variable is a variable that assosiates with a class itself and not its instances.
There is only one copy of class variable.

SYNTAX:
ClassName.VariableName
