23. Date predifined class - time


 

Some methods of Date

Methods Explanation
Date() Returns a new Date object
getHours() Returns the hours from a Date object. (0-23)
getMinutes() Returns the minutes from a Date object. (0-59)
getSeconds() Returns the seconds from a Date object. (0-59)
getTime() Returns a Date in milliseconds
