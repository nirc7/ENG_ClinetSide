14. Array - the basic data structure:

The JS Array has the advantages of both array and linked list in other languages. As JS is not typed, it can even serve as a struct, or list of structs.

Array has the following characteristics:
immediate access -
in O(1) time to any element
dynamic -
the length is increased as needed
automatic initialization -
all the undefined elements are initialized to "undefined"
gaps allowed -
it is possible to init any element, even if previous have not been declared or initialized, or even outside the length
automatic typing -
all are treated as strings and converted as needed
