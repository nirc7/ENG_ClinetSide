Lifetime of Variables

Local variable

A variable within a function which is declared as is VAR is a local variable and can only be accessed only within that function.
When you exit the function, the variable is destroyed. ALL other varialbes are global.

Global variable

If you declare a variable outside a function then it is global variable.
All the functions on your page can access it.
The lifetime of these variables starts when they are declared, and ends when the page is closed.
