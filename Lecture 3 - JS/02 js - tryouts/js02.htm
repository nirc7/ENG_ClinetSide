2. The #include program:

The code can be embedded in HTML in two ways:
directly -
by listing the code inside SCRIPT element
importing -
by listing the code in another JS file and calling inside the SCRIPT element the JS file

The JS file is any text file, including only the JS code and having the .js name ending.

The including of the imported file is using the SRC property of the SCRIPT tag:

<script src="filename.js"><script>

The code does not have to be imported at the beginning of the HTML file. The code is performed in the place where it is inserted.
