18. Objects, Constructors and Variables


 

Constructor

If you wish to create an object different from predefined JS objects than you have to write a constructor for this object.

  • First you write your constructor using
    function ClassName(Arglist)
    {
    // initialization using this operator
    }
     
  • you invoke your constructor by using new operator
    variable = new ClassName(Arglist);
     
  • constructor creates a new instance of your object and performs an appropriate initialization of it
