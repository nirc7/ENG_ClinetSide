11. Input and Output:

The special output features are:
  • to a document(frame) - document.write(string)
  • to a dialog pop-up window -
    • alert(string)
    • prompt(string)
    • confirm(string)
  • to an object (usually to a form) -
    • visual or text
    • assigning or changing a property
