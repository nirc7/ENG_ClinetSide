19. Comments and paragraph:

You can add comments to your page. Comment is a text that provides explanatory information, usually about the function of the code. Comments are usually helpful to someone maintaining or enhancing your page when you are no longer around to answer questions about it.

Comment syntax is: <!-- some text -->.
Text written inside comment's opening tag <!-- and comment's closing tag --> is invisible to the user.

Web browser ignores all of the CARRIAGE RETURNS typed into your text editor:

  • new lines,
  • tabs,
  • white spaces.
But wherever a browser sees the <p> tag, it inserts a blank line and starts a new paragraph.
