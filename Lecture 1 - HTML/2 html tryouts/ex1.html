1. The minimal program:

This is a minimal program.

A tag is an expression that starts with   <    and ends with   >.

A document is everything that is between opening HTML tag ( <html> ) and closing HTML tag ( </html> ).

A Document consist of the two distinct parts:

  • head
  • body

Strictly all the tags are needed. But browser are forgiving and no tag of those strictly formal is mandatory.

When those tags are obligatory?

When we need to use a tag that has to be inside one of the elements: HTML, HEAD or BODY; or an attribute that has to be inside one of those tags (usually BODY).
