5. Formatted output - predefined font styles:

Sometimes those predefined styles are called inline font styles.
The tags we used are:
  • <b> - for bold text,
  • <i> - for italics text,
  • <u> - for underlined text.
You can merge tags together. For example, if you want to write bold and underlined text you will use <b><u> combination then you should write your text and close your tags in correct order </u></b>.

<br> tag stands for break, it adds a new line to your text.
