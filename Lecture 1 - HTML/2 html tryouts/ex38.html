38. Form - input

Form input type="text" has several attributes:
  • value - string written inside text box, initial value of the input field
  • size - the length of the input field, if the user enters more characters, the text will scroll
  • name - user defined name of the current form element
Main name usage - event handling. 
In current example we used name attribute to refer to concrete input box and  to define new foreground and background colors.
<input type="text" value="blue" name="i"><br>
<input type="text" name="bgc" onChange="document.bgColor=bgc.value ; document.fgColor=i.value">
where onChange is event to be run when the element changes.

If you want to perform several actions on some event you write action after action with ; as separation mark.
onChange="document.bgColor=bgc.value ; document.fgColor=i.value"
