15. Graphical and Predefined Color Attributes - body attributes:


Body attribute background=image - puts image to be a back ground of the document.
Body attribute text=color - changes the default black text color of the document to the color you mentioned.

You can put an image,a picture to background of your page. This image will become a wallpaper of the current document.
You add background image by adding background attribute to the body tag:
<body background="pictureName.gif">.

You can change the text color of the whole document. Default text color is black, but you can redefine it by adding text attribute to the body tag:
<body text="red">.
