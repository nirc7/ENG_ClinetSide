6. Formatted output - predefined phrase styles:


Sometimes those predefined styles are called inline phrase styles.
Those tags are additional formatting tags.
  • <tt> - for teletype text (old typing machine style),
  • <s> and <strike> - for striked text,
  • <strong> - for strong, bolded text,
  • <code>, <kbd> and <sample > -  for code examples, more wide letters.
