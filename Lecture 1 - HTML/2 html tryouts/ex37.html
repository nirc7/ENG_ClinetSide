37. Form - input

A form is an area that can contain form elements. Forms are the primary input output device for web pages.

Form elements are elements that allow the user to enter information :

  • text fields,
  • textarea fields,
  • drop-down menus,
  • radio buttons,
  • checkboxes

A form is defined with the <form> tag. The most used form tag is the <input> tag. The type of input is specified with the type attribute. The most commonly used input types are:

  • text
  • password
  • button
  • checkbox
  • radio
  • file
