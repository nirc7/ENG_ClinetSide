27. Hypermedia:

Hypermedias most outstanding ability is the link. Linking is the possibility to jump from a place in a document to another "linked" place by simple click of the mouse.

Linking is of two kinds:
  • to the same document - to an "anchor" in the document
  • to another document - making the browser to load it
The linking is implemented using anchors (<a> elements). The anchor syntax is very simple:
  1. <a followed by space
  2. href="filename" followed by > (stands for hyper reference)
  3. text or image to become a link
  4. </a> closing anchor tag
<a href="filename">link text</a>

When the jump is to local file in the same folder, directory then you just write the file name: <a href="ex21.html">text</a>
When the file is in a sub-folder of the current folder: <a href="subfolder/filename">text</a>
When the file is in a parent folder of the current folder: <a href="../filename">text</a>

Local link the same document

The anchor at the start of the jump is implemented using:
<a href="#where to jump">.

When a special place in a document is the target, the anchor at the target of the jump is implemented using:
<a name="where to jump"> ... <a>.

Example of an anchor:

Go to Top

Code of this anchor:
  1. At the top of this document an anchor was created using this code: <a name="top"></a>
  2. At the bottom the link was created using the following code: <a href="#top">Go to Top</a>

Local link another document

If the jump is to another document, and no special place in it is indicated, there is no need for the second anchor. The jump will be implemented by just loading the target document (if we wish - a default anchor is implied at the top of the loaded document).
Example: <a href="ex21.html">text</a>

Absolute global addresing

In global link you have to give URL of the file: scheme://host.domain/path/ filename
<a href="URL">link text</a>
Example:
<a href="http://www.cnn.com">Go to CNN homepage </a>
