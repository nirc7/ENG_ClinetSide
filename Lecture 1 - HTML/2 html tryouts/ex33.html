33. Event handlers - DoubleClick:

Handling Events

HTML applications should be largely event-driven.
Events are actions that occur usually as a result of something the user does. For example, clicking a button is an event, as is changing a text field or moving the mouse over a link. For your script to react to an event, you define event handlers, such as onChange and onClick.
