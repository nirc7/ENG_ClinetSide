46.Grouping elements using <DIV> and <SPAN> tags

  • <DIV> tag is used to divide your text into parts. The important attribute of <DIV> tag is alignment. The text inside the <DIV> tags is put on a new line.
  • <SPAN> tag is often used in CSS (cascading style sheets).<SPAN>tag is inline tag.If you want to provide style information solely for a few words, or even letters, the best approach is to use the <SPAN> element.
  • We use <DIV> and <SPAN> to achieve the desired structural and presentational effects.
