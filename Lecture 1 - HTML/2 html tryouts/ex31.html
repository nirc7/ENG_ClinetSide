31. Multimedia - Video:

This video tag will work only in Microsoft Explorer.
You add video file to your page by using <img> tag and dynsrc="FileName" attribute: <img dynsrc="FileName" loop=infinite>
loop attribute values can be from 1 to infinite. If you want to play the file only once you can omit the loop attribute.

Another tag having the same effect: EMBED e.g.:

<EMBED DYNSRC="1.avi"">
