20. Simple Table: -symmetrical table

table -
the BEGIN and END tag of the table definition
tr -
"table row" - the BEGIN and END tag of a table row definition
td -
"table data" - the BEGIN and END tag of a cell definition

Main table <table> tag attributes:
align=center|left|right - horizontal alignment of the table
border=number
cellpadding=number - distance of text from the cell borders
cellspacing=number - distance between the cells
width=number or %number
height=number or %number

Main table row <tr> and table data tags <td>attributes:
align=middle|left|right - horizontal alignment inside the row
valign=middle|botton|top - vertical alignment inside the row
width=number or %number relative to the table width
height=number or %number relative to the table height
colspan=number - number of columns the cell should span rowspan=number - number of rows the cell should span

NOTE: Table width and height can exceed 100%. But if the measures are too small to include the text they will automatically increase to the minimum needed for text visibility.
