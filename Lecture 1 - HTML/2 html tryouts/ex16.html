16. Graphical and Predefined Color Attributes - font tag

If you want to change color of only one line or one paragraph in your document you should add <font> tag to your page.

<font> tag has size and color attributes.
You can change text color by adding color attribute to the font tag:
<font color="red">.
You can change text size by adding size attribute to the font tag:
<font size=5>.
Size of the font is in range from 1 to 7. Default size is 3, 1 is the smallest font , 7 is the largest font.
You can add a numeric integer value to the default size by using following syntax:
<font size=+2>
