47. Form input - textarea:

Textarea has following attributes:
  • cols - width of the area - number of columns
  • rows - height of the area - number of rows
  • name - the name of the area.
Text written inside textarea can be retrieved using name.value attribute.

document.write(string)
- is a function that writes string text into the display of an HTML page.  Button event handler :
onclick=document.write(ta.value)
takes value of the textarea named "ta" and executes it (writes its code to the page").
