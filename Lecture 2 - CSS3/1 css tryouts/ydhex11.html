11. CSS - background-image positioning (2):


To set a starting position of your background image
background-position property should be used.

Values of background-position can be:

  • top left
  • top center
  • top right
  • center left
  • center center
  • center right
  • bottom left
  • bottom center
  • bottom right
  • x-% y-%
  • x-pos y-pos

If you want to repeat your image background-repeat property should be used.
It can have the following values:

  • repeat
  • repeat-x
  • repeat-y
  • no-repeat
