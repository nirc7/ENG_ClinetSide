15. CSS - spacing property:


The letter-spacing property increases or decreases the white space between characters.
Values of letter-spacing property can be:
  • normal - defines normal space between characters
  • length - defines a fixed space between characters
where a fixed space can have negative values.

The line-height property increases or decreases the line height.
Values of line-height property can be:

  • normal - defines normal space
  • number - the line height is calculated by multiplying the element's font size by the number
  • percentage - the line height is calculated relatively to the element's font size (negative values are not permitted)

Length

A length value is formed by an optional + or - followed by a number and by a two-letter abbreviation that indicates the unit.
There are no spaces in a length value; e.g., 1.3 cm is not a valid length value, but 1.3cm is valid.
Length units can be relative and absolute ones.

Absolute lenght units are:

  • in (inches; 1in=2.54cm)
  • cm (centimeters; 1cm=10mm)
  • mm (millimeters)
  • pt (points; 1pt=1/72in)
  • pc (picas; 1pc=12pt)

Relative lenght units are:

  • em (ems, the height of the element's font)
  • ex (x-height, the height of the letter "x")
  • px (pixels, relative to the canvas resolution)
