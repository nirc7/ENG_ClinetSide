5. CSS - Creating a tag Style:


Pattern style is a rule of formatting for an HTML element or elements.

Each pattern driven style definition consists of:

  • Selector - tag/tags to apply to
  • Rules definitions - a series of rules, each of the form:
    property:value;
Each rules definition group is delimited by curly braces.{ }
Between the rules in the definition the semicolon (;) serves as delimiter.

As selector we use HTML tags or combinations of them.
Between the pattern no other delimiter is needed.

Though CSS is case sensitive, the selectors (being HTML case insensitive tag names) are case insensitive.
