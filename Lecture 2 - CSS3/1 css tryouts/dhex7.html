7. CSS - Creating an ID class Style:


ID style is a rule of formatting, not attached to an HTML element or elements.
Each ID driven style definition consists of:
  • ID name - #name
  • Rules definitions - same as for patterns, i.e.: a series of rules, each of the form:
    property:value;
Each rules group for one ID is delimited by curly braces.{}
Between the rules in the definition the semicolon (;) serves as delimiter.

ID name in the class declaration starts with a hash (#).

The id attribute has to be unique on the page.
There can only be one element with a given id in a document.
