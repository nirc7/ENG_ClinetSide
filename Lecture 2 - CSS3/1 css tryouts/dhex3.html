3. CSS - Creating a <LINK> StyleSheet:


The dhex3.css file listing:


   BODY      { background-color:#EEEEEE; 
	       color: red                }

   H1     {  font: 18pt Arial bold;
             color:blue;                 }

Styles are normally saved in external to your HTML documents files.
CSS file has a .css ending.

The file is included by using:

<LINK rel=stylesheet href="filename.css" type="text/css">
or <style>@import url("filename.css");</style>

External style sheets enable you to change

  • the appearance
  • the layout
of all the pages in your Web just by editing a single CSS document.
