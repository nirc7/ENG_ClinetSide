16. CSS - text-decoration property:


The text-decoration property decorates the text.
Values of text-decoration property can be:
  • underline - defines a line under the text <u>
  • overline - defines a line over the text
  • line-through - defines a line through, striked text <strike>
  • none - defines a normal, non decorated text
