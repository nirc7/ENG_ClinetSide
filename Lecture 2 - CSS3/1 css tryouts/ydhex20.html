20. CSS - font property:


The Font properties allow you to change
  • the font family
  • weigth, boldness
  • size
  • the style of a text.

Font properties:

font-family property:
a prioritized list of a specific font name or a generic font family for an element. The browser will use the first value it recognizes. Some generic-family names are:

  • serif (e.g., Times)
  • sans-serif (e.g., Arial or Helvetica)
  • cursive (e.g., Zapf-Chancery)
  • fantasy (e.g., Western)
  • monospace (e.g., Courier)

font-weight property:
sets how thick or thin characters in text should be displayed. Possible values are:

  • normal
  • bold - defines thick characters
  • bolder - defines thicker characters
  • lighter - defines lighter characters
  • 100,200,300,400-900 - defines from thin to thick characters. 400 is the same as normal, and 700 is the same as bold.

font-size property:
sets the size of a font. Possible values are:

  • xx-small,x-small,small,medium,large,x-large,xx-large - sets the size of the font to different sizes, from xx-small to xx-large (as clothes sizes)
  • smaller - sets the font-size to a smaller size then the parent element
  • larger - sets the font-size to a larger size then the parent element
  • length - sets the font-size to a fixed length size

font-style property:
sets the style of a font. Possible values are:

  • normal - browser displays a normal font
  • italic - browser displays an italic font <i>
  • oblique - browser displays an oblique font
