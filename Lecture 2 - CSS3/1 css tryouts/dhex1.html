1. CSS - Creating a <STYLE> StyleSheet:

DHTML = HTML + JavaScript + CSS

  • CSS - Cascading Style Sheets
  • Styles define how to display HTML elements
  • Styles are normally stored in Style Sheets
  • Styles were added to HTML 4.0 and were supproted by Netscape 4.0 and Internet Explorer 4.0

Styles can be specified inside

  • a single HTML element
  • inside the <head> element of an HTML page
  • or in an external CSS file
