6. CSS - Creating a class Style:


Class style is a rule of formatting, not attached to an HTML element or elements.

Each class driven style definition consists of:

  • Class name - . name
  • Rules definitions - same as for patterns, i.e.: a series of rules, each of the form:
    property:value;
Each rules group for one class is delimited by curly braces. {}
Between the rules in the definition the semicolon (;) serves as delimiter.

Class name in the class declaration starts with a dot.

Between the classes no other delimiter is needed.

More than one element can be declared to belong to one or more classes.
