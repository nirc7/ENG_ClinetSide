6a. CSS - Cascading Order


What style will be used when there is more than one style
specified for an HTML element?

All the styles will "cascade" into a new "virtual" Style Sheet.
As in
<p class="big-blue bgtomato caps">this is a para belonging to three classes: big-blue and bgtomato and caps!</p>

In case of collisions the following highest priority rules will be applied:

  1. Browser default
  2. External Style Sheet
  3. Internal Style Sheet (inside the <head> tag)
  4. Inline Style (inside HTML element)

where an inline style (inside an HTML element) has the highest priority.
It will override every style declared inside

  • the <head> tag
  • in an external style sheet
  • and in a browser (a default value).
