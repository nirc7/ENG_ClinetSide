22. CSS - border:


The Border properties allow you to set:
  • width
  • style
  • color
of an element's border.
In HTML we used tables to create borders.

Border properties

border-style property
sets the style of the four borders. Can have from one to four styles. Possible Values:

Value Description
none Defines no border
hidden The same as "none", except in border conflict resolution for table elements
dotted Defines a dotted border. Renders as solid in most browsers
dashed Defines a dashed border. Renders as solid in most browsers
solid Defines a solid border
double Defines two borders. The width of the two borders are the same as the border-width value
groove Defines a 3D grooved border. The effect depends on the border-color value
ridge Defines a 3D ridged border. The effect depends on the border-color value
inset Defines a 3D inset border. The effect depends on the border-color value
outset Defines a 3D outset border. The effect depends on the border-color value

border-width property:
sets the width of the four borders in one declaration, can have from one to four values.Possible values:

  • thin - defines a thin border
  • medium - defines a medium border
  • thick - defines a thick border
  • length - allows to define the thickness of the borders.
